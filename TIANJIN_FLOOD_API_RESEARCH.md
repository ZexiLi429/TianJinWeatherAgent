# 天津市积涝防洪水位监测API数据源调研报告

## 📋 执行摘要

经过全面调研，天津市的积涝、防洪、水位监测数据主要来源于以下几个官方和第三方API，但**直接的公开实时数据API较为有限**。大多数数据需要通过官方部门申请或通过气象部门的公开数据获取。

---

## 🔍 调研结果总结

### ✅ 可获取的真实API数据源

#### 1. **和风天气 QWeather (推荐：可获取暴雨预警数据)**

**是否有积涝数据：** ⚠️ 不直接提供积涝数据，但提供暴雨预警
**官网：** https://www.qweather.com/
**API文档：** https://dev.qweather.com/docs/api/

**获取暴雨预警数据（间接指示积涝风险）：**

```
POST https://api.qweather.com/v7/warning/now

参数：
  location=天津市     (可用城市代码或坐标)
  key=YOUR_API_KEY    (需要免费注册获取)

返回数据示例：
{
  "code": "200",
  "updateTime": "2026-05-17T14:30+08:00",
  "fxLink": "https://www.qweather.com/...",
  "warning": [
    {
      "id": "202605171430001",
      "type": "12",           // 12=暴雨
      "typeName": "暴雨",
      "level": "3",           // 风险等级: 1=蓝色 2=黄色 3=橙色 4=红色
      "levelName": "橙色",
      "text": "天津市和平区将在12小时内出现强降水，易引发积涝",
      "startTime": "2026-05-17T14:00+08:00",
      "endTime": "2026-05-17T20:00+08:00",
      "effective": "2026-05-17T14:30+08:00",
      "expire": "2026-05-17T21:00+08:00"
    }
  ]
}
```

**申请步骤：**
1. 访问 https://dev.qweather.com/
2. 注册账户
3. 创建应用获取 API Key
4. 免费版支持请求数/天的限制

**更新频率：** 实时（警告通常每6-12小时更新）
**是否需要API密钥：** ✅ 需要（免费版本即可）
**数据更新频率：** 实时～6小时

#### 2. **高德天气API (High Maps Weather)**

**是否有积涝数据：** ⚠️ 提供降雨数据，可间接判断积涝风险
**官网：** https://lbs.amap.com/
**API文档：** https://lbs.amap.com/api/webservice/guide/api/weatherinfo/

**获取天气和降雨数据：**

```
GET https://restapi.amap.com/v3/weather/weatherInfo

参数：
  city=120100           (天津市代码)
  key=YOUR_API_KEY      (需要免费注册)

响应示例：
{
  "status": "1",
  "count": "1",
  "info": "OK",
  "infocode": "10000",
  "lives": [
    {
      "province": "天津",
      "city": "天津市",
      "adcode": "120100",
      "weather": "小雨",
      "temperature": "18",
      "winddirection": "南",
      "windpower": "3",
      "humidity": "89",
      "reporttime": "2026-05-17 14:30:00"
    }
  ]
}
```

**申请步骤：**
1. 访问 https://lbs.amap.com/
2. 注册并认证高德账户
3. 创建 Web 服务应用
4. 获取 API Key

**是否需要API密钥：** ✅ 需要
**数据更新频率：** 每小时更新

---

#### 3. **中国气象局 - 预警服务 (CMSGSV)**

**是否有积涝数据：** ✅ 提供暴雨预警信息
**官网：** http://www.cma.gov.cn/
**数据源：** http://www.nmc.cn/publish/forecast/

**获取天津市气象预警：**

```
GET http://www.nmc.cn/publish/forecast/ANE/tianjin.html
或
GET http://www.nmc.cn/rest/queryalarm (非官方REST API)

说明：中国气象部门的数据采用HTML页面爬取方式
实际可用的机构API：http://api.nmc.cn/ (部分服务)
```

**直接查询示例：**
```
浏览器访问：
http://www.nmc.cn/publish/forecast/ANE/tianjin.html
```

**是否需要API密钥：** ⚠️ 部分API需要申请，部分数据公开
**数据更新频率：** 实时（警告发布即时）

---

#### 4. **天津市应急管理部门 - 预警平台**

**是否有积涝数据：** ✅ 有，但大多数数据非公开API形式
**官网：** https://www.tj.gov.cn/ (天津市人民政府)
**相关部门：** 
- 天津市水务局：https://swj.tj.gov.cn/
- 天津市气象部门：https://tj.cma.gov.cn/

**获取防洪预警信息：**

```
天津市水务局官网可查看：
https://swj.tj.gov.cn/

相关页面：
- 防汛预报：通常在汛期（6-9月）发布
- 水位数据：部分实时数据可从页面获取
- 预警信息：通过RSS订阅或定期查看
```

**是否需要API密钥：** ❌ 数据本身公开，但需要部门授权获取API
**数据更新频率：** 根据汛期情况，每6-12小时或实时

## 🚨 关键发现：天津市积涝数据的现状

### 问题分析

1. **缺乏专门的积涝实时API**
   - 中国暂无国家级的实时积涝监测开放API
   - 积涝数据主要由城市排水部门和应急部门内部使用

2. **可获取的间接数据**
   - ✅ 暴雨预警（可推断积涝风险）
   - ✅ 降水量实时数据（高德、和风等）
   - ✅ 官方预警信息（中国气象局、防总）

3. **需要部门合作获取的数据**
   - 天津市水务局内部水位监测数据
   - 城市积涝监测系统数据
   - 排水管网实时数据
