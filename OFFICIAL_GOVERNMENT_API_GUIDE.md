# 天津市官方部门防洪预警API汇总

## 📋 核心官方数据源

本文档汇总天津市及中国各级政府部门提供的防洪、水位、积涝监测的官方数据接口。

---

## 🏛️ 第一级：中央政府部门

### 1. 水利部 - 全国水情数据服务

**机构：** 中华人民共和国水利部  
**官网：** http://www.mwr.gov.cn/  
**数据平台：** http://www.hydroinfo.gov.cn/

#### 📊 数据服务功能

```
官方数据服务：http://www.hydroinfo.gov.cn/
- 全国主要江河水位实时数据
- 降雨量分布图
- 水文预报预警
- 水库蓄水情况
- 防汛抗旱动态
```

#### 🔍 获取天津水情数据

**URI接口：**
```
http://www.hydroinfo.gov.cn/web/pages/index.html

可查询的重点：
- 海河流域水位情况
- 天津地区主要水文站实时数据
- 暴雨预报信息
- 防汛通报
```

#### 📡 实时数据查询

```
查看路径：
1. 访问 http://www.hydroinfo.gov.cn/
2. 选择 "流域信息" → "华北地区"
3. 在地图上查找天津相关水文站点
4. 查看实时水位、流量等数据

关键水文站（天津）：
- 海河三岔河口站
- 海河北闸站
- 永定河沿村站
- 潮白河蓝村站
- 大清河田家庵站
```

#### 🔐 API接口获取

```
非官方获取方式（通过爬取或第三方API）：
- 部分数据可通过 DataEye 等大数据平台获取
- 需要填写《水利数据获取申请表》向水利部申请
- 企业用户可申请 B2B 数据接口服务

申请流程：
1. 访问 http://www.mwr.gov.cn/
2. 进入 "办事大厅" → "网上申报"
3. 填写"水文数据获取申请"
4. 提供企业资质和用途说明
5. 等待审批（通常需要7-30天）
```

#### 📍 天津相关的主要水文站

| 水文站名 | 所在河流 | 用途 | 坐标 |
|---------|--------|------|------|
| 三岔河口 | 海河 | 防洪预警 | 39°05'N 117°11'E |
| 北闸 | 海河 | 水位监测 | 39°03'N 117°12'E |
| 沿村 | 永定河 | 洪水预警 | 39°50'N 116°00'E |

| 田家庵 | 大清河 | 流量监测 | 38°40'N 116°00'E |

### 2. 中国气象部门 - 预警信息

**机构：** 中国气象局（CMA）  
**官网：** http://www.cma.gov.cn/  
**数据平台：** http://www.nmc.cn/ (国家气象中心)  
**预警中心：** http://www.nmc.cn/publish/alarm/

#### 📊 获取天津市预警信息

**官方页面：**
```
http://www.nmc.cn/publish/forecast/ANE/tianjin.html

包含信息：
- 预报参数 (温度、降水等)
- 灾害天气预警
- 强对流天气预警
- 气象灾害风险预警
```

**预警查询：**
```
所有全国预警：
http://www.nmc.cn/publish/alarm/

特别关注预警类型：
- 暴雨预警
- 强对流预警
- 分布式暴雨预警
```

#### 🔐 预警信息获取方式

##### 方式1：手动查询（即时）
```
访问：http://www.nmc.cn/publish/alarm/
在地图上点击天津市查看当前预警
```

##### 方式2：RSS订阅（每小时更新）
```
RSS订阅链接：
http://www.nmc.cn/f/fm/rss/forecast_tianjin.html

使用RSS阅读器订阅，实时获取天津预报信息
```

##### 方式3：官方API（通常需要特殊权限）
```
中国气象部门的 REST API：
http://api.nmc.cn/ (部分服务)

需要联系中国气象部门申请权限：
- 企业API协议
- 学校/科研机构特殊申请
```

##### 方式4：通过授权的气象数据服务商
```
例如：
- Alexa Weather API
- OpenWeatherMap (包含中国数据)
- 和风天气 (与CMA合作)
- 墨迹天气 (与CMA合作)
```

#### 📡 天津气象部门直接服务

**天津市气象部门：** https://tj.cma.gov.cn/

```
官方服务：
1. 预警短信订阅
   - 发送"预警"到 12345 获取预警信息

2. 预警电话：022-2386xxxx (需查询最新)

3. 官方微信/微博
   - 扫码关注"天津气象"公众号
   - 获取实时预警和预报

4. 预警平台
   - https://tj.cma.gov.cn/gj/index.html
```

## 🎓 数据申请模板

### 向官方部门申请API访问权限的标准流程

```
第1步：准备申请材料
- 企业/机构营业执照或证明
- 项目介绍和计划书
- 数据用途说明
- 预期受众和影响范围
- 数据保密承诺

第2步：递交申请
- 邮件或在线表单
- 抄送相关部门领导
- 清晰标注数据需求
- 留下有效联系方式

第3步：等待审批
- 通常需要 7-30 天
- 可能被要求提供补充资料
- 积极跟进审批进展

第4步：签署协议
- 审批通过后提供协议
- 包含保密条款
- 包含使用范围限制
- 包含数据更新承诺

第5步：获得API凭证
- 获得 API Key 或数据库访问权限
- 获得使用文档
- 建立技术对接小组（大型合作）
```

## 🔐 重要提示

### 数据使用的合法性

```
✅ 允许的用途：
- 公众防灾减灾服务
- 学术研究和科学评估
- 应急决策支持
- 安全预警发布
- 公众信息教育

❌ 禁止的用途：
- 商业牟利和转卖

```

实时更新（小时级）：

定期发布（天级）：
- 天气预报

所有集成的API应处理不同的更新频率。
